# Portfolio Hedging Analysis

## Overview
This project presents a comprehensive quantitative risk management framework for an equity portfolio. 
The analysis includes beta estimation, hedge ratio calculation, futures-based hedging, and dynamic delta-neutral option strategies.

## Objectives
- Construct and analyze portfolio returns
- Estimate portfolio beta and correlation with market index
- Compute optimal hedge ratios
- Implement futures hedging strategies
- Apply dynamic delta-neutral hedging using options
- Compare hedged vs unhedged portfolio performance

## Methodology
- Return and volatility analysis
- Regression-based beta estimation
- Static and rolling hedge ratios
- Black-Scholes framework for option pricing
- Greeks calculation (Delta)
- Dynamic portfolio rebalancing

## Tools & Libraries
- Python
- NumPy
- Pandas
- Matplotlib

## Key Findings
The implementation of systematic hedging strategies significantly reduced portfolio volatility.
Dynamic delta-neutral adjustments provided improved risk control compared to unhedged positions.
